# sylmark-vscode-plugin README

This is the README for your extension "sylmark-vscode-plugin".

## Steps to test VSCode plugin locally

1. Clone the repository

```sh
git clone https://github.com/ReshmaKhot/sylmark-vscode-plugin.git
```

2. Checkout the directory

```sh
cd sylmark-vscode-plugin
```

3. Install the dependencies using following command

```sh
npm install
```

4. Open the directory with VSCode

```sh
code .
```

5. Start extension debugging using one of the options below, this will open new VSCode window with your extension running
   - Press `F5`
   - Use `start debugging` option listed under VSCode `Run` menu
6. Now you can test your extension
